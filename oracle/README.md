# Oracle

战略技术顾问 - 高难度架构设计、复杂问题调试、技术决策咨询。

## 用法

```bash
/oracle                          # 启动架构咨询
/oracle "这个系统该如何重构？"    # 具体问题咨询
```

## 核心能力

- **架构分析**：评估现有架构，提供重构建议
- **调试诊断**：定位复杂 bug、性能瓶颈
- **技术决策**：框架对比、架构权衡

## 特点

- **只读咨询**：提供分析建议，不修改代码
- **务实简洁**：优先最简单的解决方案
- **分层输出**：核心结论 → 详细分析 → 边界情况
