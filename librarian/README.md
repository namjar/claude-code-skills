# Librarian

开源库研究专家 - 查找文档、搜索 GitHub 示例、理解库的内部实现。

## 用法

```bash
/librarian "React Suspense 是如何实现的？"
/librarian "prisma 如何处理数据库迁移？"
```

## 核心能力

- **文档检索**：查找官方文档和 API 参考
- **源码分析**：深入理解库的内部实现
- **示例搜索**：查找 GitHub 使用示例

## 特点

- **必须引用来源**：所有声明需要文档或代码支持
- **GitHub 永久链接**：代码引用精确到行号
- **版本意识**：标注讨论的库版本
