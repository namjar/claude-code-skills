# Thinking Frameworks

25种跨职业思维工具框架，帮助从多角度分析复杂问题。

## 用法

```bash
/thinking-frameworks
```

当面对复杂问题、需要多角度分析、做决策、或需要创造性解决方案时使用。

## 25种思维工具

### 创造与设计类
- **艺术家**：优先创意和独特性，突破约束
- **建筑师**：创建缩放模型预见结果
- **设计师**：让物品自然传达其用途
- **小说家**：将信息结构化为连贯叙事

### 分析与验证类
- **科学家**：形成假设，系统检验
- **数学家**：要求严格证明才接受结论
- **记者**：从独立来源交叉验证事实
- **评论家**：建立分析连接作品与更广泛观念
- **会计师**：追踪比率揭示隐藏模式

### 系统与工程类
- **工程师**：建立系统模型，精确计算
- **程序员**：识别模式以自动化和简化
- **水管工**：拆解系统理解故障
- **黑客**：探索表面抽象下的底层机制

### 诊断与理解类
- **医生**：通过症状诊断根本原因
- **心理学家**：通过行为观察验证理解
- **人类学家**：沉浸在陌生环境中深入理解
- **教师**：识别知识差距并有效桥接

### 战略与预测类
- **经济学家**：理解人们如何响应激励
- **棋手**：心理推演未来多步场景
- **政治家**：计算行为将如何被他人感知
- **哲学家**：探索假设的意外后果

### 行动与执行类
- **企业家**：快速尝试多种方法发现有效方案
- **士兵**：用常规和纪律防止错误
- **销售员**：从行为模式推断未表达的需求
- **演员**：通过想象产生真实情感

## 快速参考

|------------|--------|

| 理解系统 | 工程师、黑客 |
| 预测行为 | 经济学家、心理学家 |
| 战略规划 | 棋手、政治家 |

| 理解他人 | 人类学家、销售员、演员 |
