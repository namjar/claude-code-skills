# Doc Writer

技术文档专家 - 撰写 README、API 文档、架构文档、用户指南。

## 用法

```bash
/doc-writer                        # 启动文档写作
/doc-writer --readme               # 生成 README
/doc-writer --api                  # 生成 API 文档
/doc-writer --architecture         # 生成架构文档
```

## 文档类型

- **README**：项目门面
- **API 文档**：端点、参数、响应
- **架构文档**：系统设计、数据流
- **用户指南**：操作手册

## 核心原则

> "未经验证的文档是有害的。"

- 所有代码示例必须测试
- 所有命令必须执行
- 所有链接必须验证
