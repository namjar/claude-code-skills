# Invoice Scanner

扫描目录识别各类发票（交通、住宿、餐饮等），提取关键信息并生成分类统计报告。

## 用法

```bash
/invoice-scanner ./发票文件夹
/invoice-scanner ./receipts.zip
```

## 功能特点

- **多类型识别**：市内交通、长途交通、住宿、其他四大分类
- **自动解压**：支持直接处理 ZIP 文件，自动解压并扫描
- **信息提取**：从发票图片/PDF 中提取关键字段
- **汇总报告**：生成 Markdown 格式的分类统计报告

## 支持的发票类型

- **市内交通**：打车发票、地铁票、公交票
- **长途交通**：飞机票、火车票、长途汽车票
- **住宿**：酒店住宿发票、酒店结账单
- **其他**：餐饮发票、通讯费发票、办公用品等
