# Frontend Engineer

UI/UX 设计工程师 - 即使没有设计稿也能产出专业的 UI。

## 用法

```bash
/frontend-engineer                    # 启动设计会话
/frontend-engineer "设计一个登录页面"   # 设计特定组件
/frontend-engineer --style=minimal    # 指定设计风格
```

## 设计风格

| 风格 | 特点 |
|------|------|
| Brutalist Minimal | 极简、粗野、留白 |
| Soft Luxe | 柔和、奢华 |
| Tech Crisp | 锐利、现代 |
| Playful Pop | 活泼、大胆 |

## 特点

- **设计优先**：先确定美学方向再编码
- **完整设计系统**：CSS 变量定义
- **避免俗套**：拒绝常见字体和配色
